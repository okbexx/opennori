## OpenNori

This repository uses OpenNori for repo-native agent work. Before implementation,
read `.opennori/workflow.md`, the relevant files under `.opennori/spec/`, and
the current task artifacts. Use the project Skills under `.claude/skills/` to
plan, implement, verify, deliver, and finish work through the persistent
`opennori` CLI. The approved Nori Contract, current-revision Evidence, and final
Git checkpoint define completion.
