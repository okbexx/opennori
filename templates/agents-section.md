## OpenNori

This repository uses OpenNori for repo-native agent work. Before implementation,
read `.opennori/workflow.md`, the relevant files under `.opennori/spec/`, and
the current task artifacts. The approved Nori Contract defines done; task
activity alone does not. Use the installed OpenNori Skills to plan, implement,
verify, deliver, and finish work, and do not bypass missing required evidence or
the final Git checkpoint. Run stage
commands through the persistent `opennori` CLI installed by OpenNori setup.
